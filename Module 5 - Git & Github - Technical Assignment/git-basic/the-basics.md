## Jawaban Nomor 1 - the-basics

```cli
1. mkdir git-basics
2. cd git-basics
3. touch first.txt
4. git init
5. git add first.txt
6. git commit -m "adding first.txt"
7. git log
8. touch second.txt
9. git add second.txt
10. git commit -m "adding first.txt"
11. git rm second.txt
12. git add .
13. git commit -m "removing first.txt"
14. git log
```
